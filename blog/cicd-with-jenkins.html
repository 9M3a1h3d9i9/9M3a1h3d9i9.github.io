چگونه اولین پایپ‌لاین CI/CD خود را با Jenkins ساختم؟

یک راهنمای عملی و قدم به قدم

سلام! من محمد مهدی شفیقی هستم، دانشجوی کارشناسی ارشد هوش مصنوعی. در دنیای امروز، ساختن یک مدل یادگیری ماشین تنها نیمی از مسیر است. چالش اصلی، رساندن این مدل به دست کاربر نهایی به شکلی سریع، قابل اعتماد و خودکار است. اینجاست که مفاهیم CI/CD و ابزارهایی مانند Jenkins وارد میدان می‌شوند. در این مقاله، می‌خواهم قدم به قدم به شما نشان دهم که چطور...

قدم اول: ساخت یک اپلیکیشن ساده

برای شروع، یک اپلیکیشن بسیار ساده با پایتون و فریمورک Flask می‌سازیم. این اپلیکیشن یک مدل یادگیری ماشین از پیش آموزش‌دیده را بارگذاری کرده و از طریق یک API ساده، امکان پیش‌بینی را فراهم می‌کند. 

# کد فایل app.py
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "ML Model Server is running!"

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5000)

این ساده‌ترین شکل ممکن از یک اپلیکیشن فلسک است. در مراحل بعدی، منطق مدل یادگیری ماشین را به آن اضافه خواهیم کرد...
